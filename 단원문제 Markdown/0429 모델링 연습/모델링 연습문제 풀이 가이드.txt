2. 만든 데이터 프레임 "ggplot"활용하여 산점도 그래프 그림.

3. 선형회귀모델 생성 후 ggplot으로 위에서 그린 산점도 위에 모델링 그래프 추가.

4. summary함수로 회귀모델의 p-value 및 R-square 값 확인.

5. plot(model)을 통하여 4가지 그래프 확인. => 모델 진단 그래프 라고 한다.
  - 첫번째 그래프(Residual vs Fitted plot) : 종속변수와 독립변수간에 선형성을 알아볼 수 있는 그래프이다. 
  - 두번째 그래프(Normal Q-Q plot) :  정규성 확인 가능. 45도 각도의 직선 위에 각 점들이 위치한다면 정규성 가정을 만족하는 것.
  - 세 번째 그래프(Scale-Location plot) : 등분산성 확인 플롯. 점들이 랜덤한 형태로(퍼져 있어야) 등분산성을 만족
  - 네 번째 그래프(Residuals vs Leverage plot) : 이상관측치 확인 가능. 숫자가 뜨는 애들이 있으면 걔네가 이상치.
